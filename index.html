About

Translit Copt is an application that allows you to transliterate Coptic text. This application will be very useful for those learning how to read the Coptic language.

To use, allow camera access, and scan the Coptic text that you would like (or alternatively upload an image you already have), and our program will transliterate it and output it for you in the "Transliterated Text" section!
